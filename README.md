<ul>
<li>
<p>Softwares needed to be installed: Node.js, Express.js, MongoDB</p>
</li>
<li>
<p>To install Node.js: follow <a href="https://nodejs.org/en/download/package-manager/" rel="nofollow">these instructions</a>.</p>
</li>
<li>
<p>To install MongoDB: follow <a href="https://nodejs.org/en/download/package-manager/" rel="nofollow">the instructions</a> on the official website.</p>
</li>
</ul>
<h2><a href="#to-run-the-server" aria-hidden="true" class="anchor" id="user-content-to-run-the-server"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>To run the server</h2>
<ul>
<li>
<p>In the folder with server.js file, type "npm install --save"</p>
</li>
<li>
<p>type "mongod" to run the MongoDB instance</p>
</li>
<li>
<p>run "npm run start-dev" to run both ends at the same time. The front end will be listening on port 443 (default https port), and the back-end on port 1337</p>
</li>
<li>
<p>run "npm start" to run just the front-end</p>
</li>
<li>
<p>run "node server.js" to run just the back-end.</p>
</li>
<li>
<p>Hint: to run locally, remove the .env file if it exists</p>
</li>
</ul>
